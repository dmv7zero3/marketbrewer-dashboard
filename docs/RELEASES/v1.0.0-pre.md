# MarketBrewer SEO Platform — v1.0.0-pre

Release date: 2025-12-17

This pre-v1.0 release finalizes all preparation work for EC2 deployment and production testing. It delivers infrastructure-as-code, CI/CD, integration tests, performance baselines, monitoring, and operations documentation.

## Highlights

- Infrastructure-as-Code: Automated EC2 deployment with CloudFormation
- CI/CD: GitHub Actions pipeline for lint, build, tests, coverage, smoke
- E2E Integration Tests: API, keywords, service areas, jobs, pages
- Performance Baseline: Targets and SLOs for API and worker
- Monitoring: CloudWatch alarms, dashboard, metrics, and agent config
- Operations Runbook: Day-2 operations, troubleshooting, DR, and scaling

## How to deploy EC2

```bash
aws cloudformation deploy \
  --template-file infrastructure/cloudformation.yaml \
  --stack-name marketbrewer \
  --parameter-overrides \
    EnvironmentName=production \
    InstanceType=t3.large \
    KeyPairName=marketbrewer \
    EnableAutoStop=true
```

## Post-deploy checks

- Run: `npm run health:check`
- Verify dashboard (port 3002) and API (port 3001)
- Confirm CloudWatch dashboard and alarms

## Notes

- Instance: t3.large; Model: ollama llama3.2 (CPU)
- Cost target: ~$22.50/month with auto-stop enabled
