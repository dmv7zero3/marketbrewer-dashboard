## ğŸ“¦ Deployment

### System Requirements

- OS: Linux (Ubuntu 22.04+) or macOS
- Node.js: 18+
- Ollama: Installed with llama3.2 model
- Disk: 20GB minimum (for Ollama models)
- RAM: 8GB minimum, 16GB recommended

### Quick Start

```bash
# Clone repo
git clone https://github.com/dmv7zero3/marketbrewer-seo-platform.git
cd marketbrewer-seo-platform

# Install dependencies
npm install

# Seed database (optional, for testing)
npm run seed

# Start server (port 3001)
npm run server

# Start worker (in separate terminal)
npm run worker

# Start dashboard dev (port 3000)
# Legacy: Current default dashboard port is 3002.
npm run dashboard:dev

# Or build for production
npm run build:all
npm run server # Server runs production build
```
